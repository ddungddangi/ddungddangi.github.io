## 注目プロジェクト

### 1. Musubi
**概要**: 言語交換プラットフォーム開発  
**役割**: バックエンド（リアルタイムチャット・AI機能専任）  
**技術**: Java, Spring Boot, WebSocket, MySQL, Redis, Grok AI  
**内容**: WebSocket を活用したリアルタイムチャット機能を設計・実装し、チャットルーム管理とメッセージ永続化を開発しました。さらに、Grok AI と連携し、AI自動応答・翻訳・文章校正機能を実装しました。  

- GitHub: https://github.com/ddungddangi/Language-Exchange-CHAT-
- Notion: https://www.notion.so/2ca0368c16f780c1abc8d3d1041f6653?source=copy_link

### 2. AI Interview Service
**概要**: AI模擬面接Webサービス開発  
**役割**: バックエンド  
**技術**: Django REST Framework, AWS Cognito, S3, Lambda, DynamoDB, Bedrock, WebSocket  
**内容**: データ保存・取得方式を見直し、S3・DynamoDB を活用した構成へ改善しました。その結果、ユーザー向けフィードバックの応答時間を約20秒から約5秒へ短縮しました。  

- GitHub: https://github.com/ddungddangi/AI-Interview-Coaching-Service
- Notion: https://www.notion.so/2ca0368c16f780c1abc8d3d1041f6653?source=copy_link

### 3. CodeBento
**概要**: ユーザーコード実行向けPaaSサービス開発  
**役割**: PM  
**技術**: Python, Docker, SQS, EC2 Auto Scaling, Lambda  
**内容**: SoftBank Hackathon in Seoul の本選プロジェクトとして、PMとして要件整理、役割分担、進行管理を担当しました。サービス構成を見直し、ワーカー起動時間を約3分から約30秒へ短縮しました。  

- GitHub: https://github.com/SOFTBANK-Hackaton-Final-Pink/Bento-Worker
- Notion: https://www.notion.so/2ca0368c16f780c1abc8d3d1041f6653?source=copy_link
